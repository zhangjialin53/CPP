时间限制	1000ms	空间限制	256MB

类和对象的使用-3-对象数组练习
建立一个对象数组，内置五个学生的数据（学号，成绩），用指针指向数组首元素，然后输出第 1、3、5 个学生的数据。

TO-DO
注意：

每个学生的信息从键盘读入。
请使用指针的方式操作数组。
输入数据说明
一共 5 行，每行 2 个整数，分别代表每个学生的学号以及成绩。
不需要考虑输入错误，所有整数类型为 int32。
输出格式说明
应至少给出第 1, 3, 5 个学生的两个信息。
程序应给出相应的人性化提示。
测试样例
输入：

12249 66 
50788 55 
52937 80 
65386 70 
38625 24
输出：

请输入第1个学生的学号和成绩:
请输入第2个学生的学号和成绩:
请输入第3个学生的学号和成绩:
请输入第4个学生的学号和成绩:
请输入第5个学生的学号和成绩:
第1个学生的学号是 12249, 成绩为 66
第3个学生的学号是 52937, 成绩为 80
第5个学生的学号是 38625, 成绩为 24

#include <iostream>
using namespace std;

// 定义学生类
class Student {
public:
    int id;      // 学号
    int score;   // 成绩

    // 构造函数
    Student() : id(0), score(0) {}

    // 函数，用于输入学生的信息
    void setInfo(int id, int score) {
        this->id = id;
        this->score = score;
    }
};

int main() {
    // 创建5个学生的数组
    Student students[5];

    // 用指针指向数组首元素
    Student* p = students;

    // 输入5个学生的信息
    for (int i = 0; i < 5; i++) {
        int id, score;
        cout << "请输入第" << i + 1 << "个学生的学号和成绩: ";
        cin >> id >> score;
        (p + i)->setInfo(id, score);  // 使用指针操作数组
    }

    // 输出第1、3、5个学生的信息
    cout << "第1个学生的学号是 " << p[0].id << ", 成绩为 " << p[0].score << endl;
    cout << "第3个学生的学号是 " << p[2].id << ", 成绩为 " << p[2].score << endl;
    cout << "第5个学生的学号是 " << p[4].id << ", 成绩为 " << p[4].score << endl;

    return 0;
}
